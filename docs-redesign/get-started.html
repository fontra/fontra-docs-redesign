What is Fontra, in one minute

Fontra is a font editor with two parts: a client that runs in your web browser, and a server that runs on your own machine (via Fontra Pak) or on a remote host. The browser is just the interface — your files stay with you, and you can work fully offline. Put the server on a network and several people can edit the same font in real time.

It is variable-first: multiple masters, live interpolation, and variable components are foundations, not add-ons. If you've only ever made static fonts, the first-font tutorial is the fastest way to feel the difference.

Coming from another editor?

If you already use Glyphs or RoboFont, start with the migration guides instead — they map your existing vocabulary onto Fontra and show you how to try it without leaving your current tool.
